HTML Table

Historical layouts

In the days before smartphones and tablets, websites were typically built using tables. This allowed for structured fixed layouts.

But it is hard to re-format a table's layout, so developers moved away from this technique once accessing web pages from different sized devices became the norm.

HTML Table

Modern day usage

The move away from tables for layout purposes has meant that tables can be used for what they were originally intended to be used for...

...presenting tabular data!

HTML Table

Table basics

Tables are structured using columns and rows in order to present data to the user.

The core HTML elements for a table are <table>, <tr> (table row) and <td> (table data).

Table rows contain the table data elements, which make up the columns.

HTML Table

The first element in any HTML table is the <table> element. This wraps all other table-related HTML elements.

HTML Table

<tr> tags then define each table row.

Within the <tr> tags sit the <td> tags, which each define a new column.

Corresponding <td> tags within a row sit in the same column.

Table headings

If needed, headings can be added to provide context to the data. The <th> tag is used for this and they sit within a row.

HTML Table

Table caption

To add a caption or title to a table, the <caption> element can be placed just inside the opening <table> tag.

Table structure

More structure can be added inside the <table>, with the addition of <thead>, <tbody> and <tfoot> tags.
